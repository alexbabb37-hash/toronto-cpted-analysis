# Toronto CPTED Analysis
### AI-Assisted Crime Prevention Through Environmental Design

A data-driven analysis of Toronto assault patterns using open city data, 
built to identify environmental design factors contributing to crime concentration.

## What This Project Does
- Maps 254,000+ assault incidents across Toronto (2014-present)
- Analyzes crime by location type, time of day, and neighbourhood
- Overlays street lighting infrastructure against crime hotspots
- Identifies areas where environmental design interventions would reduce crime
- Deep dives into Moss Park — Toronto's highest crime neighbourhood

## Key Findings
- Apartments and outside spaces account for the majority of assaults
- Nighttime crime collapses into specific corridors suggesting lighting and access control failures
- Moss Park has adequate pole infrastructure but 6,322 assaults — indicating deeper CPTED failures beyond lighting
- Primary interventions needed: access control, territorial reinforcement, mixed land use

## Data Sources
- Toronto Major Crime Indicators: data.toronto.ca
- Toronto Topographic Mapping (Poles): data.toronto.ca
